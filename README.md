## 1、搭建
`npm info vite versions` 列出所有版本

## 2、查看结构
删除目前无用的目录和文件（css 等），使用 `VS Code` 插件 `npm Intellisense` 锁定库版本

使用 `npm config set save-prefix=""`, `pnpm config set save-prefix=""` 配置新增版本也不会带尖尖 `^`

调整、删除部分代码， **永远不要使用默认导出**
